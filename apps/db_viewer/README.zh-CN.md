# 数据库查看系统

语言: [English](README.md) | 中文说明

这是一个本地只读网页工具，用于查看 Demo SQLite 数据库 `data/futures.db`。

## 启动方式

在项目根目录运行：

```powershell
python run_db_viewer.py
```

也可以双击：

```text
open_database_viewer.bat
```

然后在浏览器打开：

```text
http://127.0.0.1:8765
```

## 功能

- 数据库总览：查看表数量、日期范围和最近 dry-run 邮件记录。
- 数据表浏览：支持筛选、分页和导出当前筛选结果。
- 品种视图：查看日线、主力映射、仓单、持仓和涨跌停等数据。
- 发送记录：查看报告级和收件人级发送记录。
- 数据质量：检查核心期货表是否覆盖目标日期。

## 说明

- SQLite 数据库以只读模式打开。
- Demo 数据为模拟数据，不适合用于交易决策。
- 页面顶部保留 `马超 / mach` 作者标识，方便公开展示时识别归属。
